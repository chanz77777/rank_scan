# ドラッグアンドドロップ機能 - v1.2

## ✨ 実装済みの改善

### 🎯 ドラッグアンドドロップ機能

R6 Siege Stats Dashboard に**完全なドラッグアンドドロップ対応**を追加しました。

---

## 🖱️ 使用方法

### パターン1: ドラッグアンドドロップ（最速）

```
1. R6 Siegeでスクリーンショットを撮る
2. ファイルエクスプローラーから画像をドラッグ
3. ページの指定エリアにドロップ
4. 自動的にプレイヤー情報が表示される ✨
```

**処理時間**: 1-3秒

### パターン2: ファイル選択（代替手段）

```
1. 「📂 ファイルを選択」ボタンをクリック
2. 画像ファイルを選択
3. 自動的に解析が開始される
```

### パターン3: 手動解析

```
1. ファイルを選択/ドロップ
2. 「🔍 解析」ボタンをクリック
3. 結果を確認
```

## 📚 コード例

### ドラッグアンドドロップハンドラ

```typescript
const handleDrop = (e: React.DragEvent<HTMLDivElement>) => {
  e.preventDefault();
  e.stopPropagation();
  setIsDragActive(false);

  const files = e.dataTransfer.files;
  if (files && files.length > 0) {
    const file = files[0];
    if (file.type.startsWith('image/')) {
      setImageFile(file);
      setImagePath(file.name);
      // 自動処理
      processImageFile();
    }
  }
};
```

### HTML マークアップ

```tsx
<div
  onDragEnter={handleDragEnter}
  onDragLeave={handleDragLeave}
  onDragOver={handleDragOver}
  onDrop={handleDrop}
  className={isDragActive ? 'border-blue-500 ring-2' : 'border-slate-700'}
>
  {isDragActive ? (
    <p>📥 ここにドロップしてください</p>
  ) : (
    <p>🖼️ スクリーンショットをドラッグ&ドロップ</p>
  )}
</div>
```

## ⚠️ トラブルシューティング

### ドラッグアンドドロップが反応しない

**確認項目:**
1. ブラウザが最新版か確認
2. JavaScript が有効になっているか確認
3. コンソール(F12)でエラーがないか確認

**解決方法:**
```typescript
// イベントの preventDefault が必要
e.preventDefault();
e.stopPropagation();
```

### ファイルが認識されない

```typescript
// MIME タイプをチェック
console.log(file.type); // "image/jpeg" など
// image/ で始まることを確認
```
