## 🚀 使用方法

### デフォルト画像で即座に確認

1. ページを開く
2. 「解析」ボタンをクリック
3. モックデータが表示される

### カスタム画像を使用

1. 「🖼️ スクリーンショットを選択」をクリック
2. ゲームのスクリーンショットを選択
3. 「🔍 解析」をクリック
4. プレイヤー情報が自動抽出される

## ❓ よくある質問

**Q: モックデータはいつ本物に変わりますか？**
A: IMAGE_PROCESSING.md のガイドに従って、Claude API を実装すると自動抽出できます

**Q: オフラインで動作しますか？**
A: Tesseract.js を使用すればオフライン動作可能です（IMAGE_PROCESSING.md 参照）

**Q: Vercel でのコスト？**
A: Claude API 使用時のみ費用発生（約 $0.003/画像）
